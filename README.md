# ya7yoo7

## Installation

Install dependencies 
```sh
npm init
```
OS X & Linux:

```sh
truffle develop
compile
migrate
```
Open Another Terminal
```sh
npm run dev
```
To deploy on ropsten
```sh
deploy --network ropsten
```
To deploy on rinkeby
```sh
truffle migrate --reset --compile-all --network rinkeby
```
